📋 Estructura esperada del Excel
  • Hoja VF — datos crudos del sistema. Columnas mínimas: car, group (cliente), des (ruta), start_date, operador, worker_id.
  • Hoja Flota — carros activos. Columna: IDS (lista de IDs de vehículos vigentes). Opcional pero recomendado: filtra carros dados de baja.
  • Hoja Inputs — dos tablas paralelas independientes:
    • Cols A-F: CLIENTE + Jefe + Gerente (+ Cantidad de carros, UDN, Horas perm.)
    • Cols H-I: CLIENTE COOR (ej. CON-CONMET-F1) + COORDINADOR
    Las filas no se corresponden — cada tabla se procesa por separado.
